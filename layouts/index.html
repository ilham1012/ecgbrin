Current Project

This project aims is to develop a system for ECG monitoring with automatic Arrhythmia detection.

In this project, there are two main concern. The first one is Arrhythmia classifier model development, and the second one is the application in the system. The plan is to develop the classifier model in our workstation, then export it to be used in the single board computer, microcontroller, or mobile devices.

Proyek Saat Ini

Proyek ini bertujuan untuk mengembangkan sistem pemantauan EKG dengan deteksi Aritmia otomatis.

Dalam proyek ini, ada dua perhatian utama. Yang pertama adalah pengembangan model classifier Arrhythmia, dan yang kedua adalah aplikasi dalam sistem. Rencananya adalah mengembangkan model classifier di workstation kami, kemudian mengekspornya untuk digunakan di komputer papan tunggal, mikrokontroler, atau perangkat seluler.
